# SeeNobi Dark Theme Color Scheme (Lighter Variant)

Based on the logo's dark background, all components use a cohesive dark color palette with lighter tones for better visibility.

## Color Palette

### Primary Colors
- **Background**: `#2d2d2d` - Main dark background (lighter than before)
- **Card Background**: `#3d4758` - Dark grey for cards (lighter)
- **Border**: `#4a5568` - Subtle borders (lighter)
- **Accent**: `#718096` - Interactive elements

### Text Colors
- **Primary Text**: `#ffffff` - Headings and important text
- **Secondary Text**: `#e8e8e8` - Body text (lighter)
- **Tertiary Text**: `#a0a0a0` - Muted text
- **Disabled Text**: `#808080` - Very muted text

### Interactive Elements
- **Primary Button**: Dark gradient `#4a5568` to `#2d3748`
- **Hover States**: Lighter shades with border highlights
- **Links**: `#718096` with hover to `#a0a0a0`

### Status Colors (Dark Theme Adapted)
- **Success**: `#2d5016` background, `#a0d4a0` text
- **Warning**: `#5a4a1a` background, `#ffd700` text
- **Info**: `#1a3a4a` background, `#7dd3fc` text
- **Danger**: `#5a1a1a` background, `#fca5a5` text

### Score Colors
- **Bronze**: `#ffd700` / `#b8860b`
- **Silver**: `#c0c0c0` / `#808080`
- **Gold**: `#ffd700` / `#ffed4e`
- **Platinum**: `#e5e4e2` / `#d4d4d4`

## Component Styling

All components follow these principles:
1. Lighter dark backgrounds with subtle borders
2. High contrast text for readability
3. Consistent spacing and padding
4. Smooth transitions and hover effects
5. Mobile-first responsive design

## Logo Integration

The logo image is displayed without text, maintaining the dark aesthetic throughout the application with improved visibility.
